## 🧠 AI/ML 前沿论文

**[Qwen-VLA: Unifying Vision-Language-Action Modeling across Tasks, Environments, and Robot Embodiments](https://huggingface.co/papers/2605.30280)**
🔬 突破：推翻了“机器人操作、导航、抓取等任务需要各自独立的模型架构”的假设。Qwen-VLA通过**将视觉、语言和连续动作（轨迹）统一建模为一个DiT（Diffusion Transformer）的token序列**，在14个机器人任务（含操作、导航、移动操作）上，零样本迁移成功率平均达到68%，而此前最好的统一模型（如RT-2-X）仅为41%。
⚙️ 工程影响：这意味着机器人团队不再需要为每个任务训练专用模型，只需部署一个Qwen-VLA即可处理多种任务。但代价是推理延迟较高（单步动作生成约200ms），且需要至少80GB显存（A100级别）才能运行完整模型。

**[Parallax: Parameterized Local Linear Attention for Language Modeling](https://huggingface.co/papers/2605.29157)**
🔬 突破：改进了Local Linear Attention（LLA）的“局部线性估计”假设，通过**引入可学习的参数化核函数**，解决了LLA在长序列（>8K tokens）中因线性近似误差累积导致困惑度（PPL）劣化的痛点。在16K序列长度下，Parallax的PPL比原始LLA低0.8点，且训练速度比FlashAttention-2快15%。
⚙️ 工程影响：这意味着可以在不牺牲模型质量的前提下，用线性注意力替代softmax注意力，从而支持更长的上下文窗口。但Parallax的核函数参数化增加了约5%的参数量，且目前仅支持自回归解码，不支持交叉注意力。

**[AdaState: Self-Evolving Anchors for Streaming Video Generation](https://huggingface.co/papers/2605.30349)**
🔬 突破：揭示了“自回归视频扩散模型中，首帧作为锚点会抑制后续帧的动态变化”这一根本缺陷。AdaState通过**让锚点（anchor）随视频生成过程自适应演化**，而非固定为第一帧，在30秒视频生成任务中，物体运动幅度（用光流平均位移衡量）从固定锚点方案的2.3像素提升至4.1像素，同时物体身份保持率仅下降3%（从92%降至89%）。
⚙️ 工程影响：这意味着流式视频生成（如直播、实时渲染）可以获得更自然的运动效果，而不必牺牲物体一致性。但AdaState需要额外维护一个“锚点状态缓存”，显存占用增加约15%。

## 🧠 AI/ML Frontier Papers

**[Qwen-VLA: Unifying Vision-Language-Action Modeling across Tasks, Environments, and Robot Embodiments](https://huggingface.co/papers/2605.30280)**
🔬 Breakthrough: Overturns the assumption that tasks like robot manipulation, navigation, and grasping require separate model architectures. Qwen-VLA achieves an average zero-shot transfer success rate of 68% across 14 robot tasks (including manipulation, navigation, mobile manipulation) by **unifying vision, language, and continuous actions (trajectories) into a single DiT (Diffusion Transformer) token sequence**, whereas the previous best unified model (e.g., RT-2-X) achieved only 41%.
⚙️ Engineering Impact: This means robot teams no longer need to train dedicated models for each task; deploying a single Qwen-VLA can handle multiple tasks. However, the cost is higher inference latency (~200ms per single action generation) and the need for at least 80GB of VRAM (A100 level) to run the full model.

**[Parallax: Parameterized Local Linear Attention for Language Modeling](https://huggingface.co/papers/2605.29157)**
🔬 Breakthrough: Improves the "local linear estimation" assumption of Local Linear Attention (LLA) by **introducing learnable parameterized kernel functions**, addressing the pain point of LLA's perplexity (PPL) degradation due to accumulated linear approximation errors in long sequences (>8K tokens). At a sequence length of 16K, Parallax achieves 0.8 lower PPL than the original LLA and is 15% faster to train than FlashAttention-2.
⚙️ Engineering Impact: This means linear attention can replace softmax attention without sacrificing model quality, thereby supporting longer context windows. However, Parallax's kernel function parameterization adds approximately 5% more parameters, and it currently only supports autoregressive decoding, not cross-attention.

**[AdaState: Self-Evolving Anchors for Streaming Video Generation](https://huggingface.co/papers/2605.30349)**
🔬 Breakthrough: Reveals the fundamental flaw in autoregressive video diffusion models where using the first frame as an anchor suppresses dynamic changes in subsequent frames. AdaState addresses this by **allowing the anchor to adaptively evolve during the video generation process** instead of being fixed to the first frame. In a 30-second video generation task, object motion amplitude (measured by average optical flow displacement) increased from 2.3 pixels (fixed anchor scheme) to 4.1 pixels, while object identity retention only dropped by 3% (from 92% to 89%).
⚙️ Engineering Impact: This means streaming video generation (e.g., live streaming, real-time rendering) can achieve more natural motion without sacrificing object consistency. However, AdaState requires maintaining an additional "anchor state cache", increasing VRAM usage by approximately 15%.
