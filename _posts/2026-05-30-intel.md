## 🧠 AI/ML 前沿论文

**[CONF-KV: Confidence-Aware KV Cache Eviction with Mixed-Precision Storage for Long-Horizon LLM](https://huggingface.co/papers/2605.24786)**
🔬 突破：推翻了一个常见假设——“KV cache 驱逐策略应基于历史注意力或时间窗口”。CONF-KV 证明，**模型在每一步解码时的“不确定性”（即 next-token 分布的熵）是比历史注意力更有效的驱逐信号**。在 128K token 的长上下文推理中，相比 H2O（基于历史注意力）和 StreamingLLM（基于时间窗口），CONF-KV 在保持相同困惑度的前提下，将 KV cache 内存占用降低 40%（从 32GB 降至 19GB）。
⚙️ 工程影响：这意味着长上下文 LLM 推理的显存瓶颈可以被显著缓解，且无需修改模型架构。团队在部署长文档问答或代码库分析服务时，可直接将 CONF-KV 集成到 vLLM 或 TensorRT-LLM 的调度器中，以降低单次推理的 GPU 成本。

**[PANDO: Efficient Multimodal AI Agents via Online Skill Distillation](https://huggingface.co/papers/2605.24785)**
🔬 突破：挑战了“AI Agent 效率提升依赖更多推理时计算”的假设。PANDO 发现，**多模态 Web Agent 的 70% 以上推理时间浪费在三种可预测的低效模式上**：重复动作循环、隐藏的探索成本、低 prompt cache 复用率。其核心创新是“在线技能蒸馏”——Agent 在每次成功完成任务后，将执行轨迹提炼为一个可复用的“技能”（一个参数化的子策略），并在后续任务中直接调用而非重新推理。在 VisualWebArena 上，PANDO 在首次任务后，后续任务的推理时间平均降低 55%，且成功率仅下降 2%。
⚙️ 工程影响：这意味着构建“越用越快”的 Agent 系统成为可能。团队在设计 Agent 框架时，应加入“技能库”组件，并设计轨迹到技能的蒸馏策略，而非仅关注单次任务的推理优化。

**[DynaFLIP: Rethinking Robotics Perception via Tri-Modal-Dynamics Guided Representation](https://huggingface.co/papers/2605.30350)**
🔬 突破：推翻了一个假设——“机器人视觉编码器应预训练于静态图像识别”。DynaFLIP 证明，**将“运动理解”前置到视觉编码器的预训练阶段，比在策略网络中从头学习运动特征更有效**。其方法是通过“图像-语言-3D 光流”三元组进行预训练，使编码器学会从单帧图像中推断物体的运动属性。在 MetaWorld 和 Franka Kitchen 基准上，使用 DynaFLIP 编码器的策略，相比使用 CLIP 编码器的策略，任务成功率平均提升 18%，且对视觉干扰的鲁棒性提升 25%。
⚙️ 工程影响：这意味着机器人学习 pipeline 需要重构：视觉编码器的预训练目标应从“识别物体”转向“理解物体如何运动”。团队在构建机器人基础模型时，应优先考虑包含运动标注的数据集（如 Bridge Data v2），而非仅使用静态图像数据集。

## 🧠 AI/ML Frontier Papers

**[CONF-KV: Confidence-Aware KV Cache Eviction with Mixed-Precision Storage for Long-Horizon LLM](https://huggingface.co/papers/2605.24786)**  
🔬 Breakthrough: Challenges a common assumption — "KV cache eviction strategies should be based on historical attention or time windows." CONF-KV proves that **the model's "uncertainty" at each decoding step (i.e., the entropy of the next-token distribution) is a more effective eviction signal than historical attention**. In long-context inference with 128K tokens, compared to H2O (based on historical attention) and StreamingLLM (based on time windows), CONF-KV reduces KV cache memory usage by 40% (from 32GB to 19GB) while maintaining the same perplexity.  
⚙️ Engineering Impact: This means the memory bottleneck for long-context LLM inference can be significantly alleviated without modifying the model architecture. Teams deploying long-document Q&A or codebase analysis services can directly integrate CONF-KV into the scheduler of vLLM or TensorRT-LLM to reduce GPU cost per inference.

**[PANDO: Efficient Multimodal AI Agents via Online Skill Distillation](https://huggingface.co/papers/2605.24785)**  
🔬 Breakthrough: Challenges the assumption that "AI agent efficiency improvement relies on more inference-time computation." PANDO finds that **over 70% of inference time for multimodal web agents is wasted on three predictable inefficiency patterns**: repetitive action loops, hidden exploration costs, and low prompt cache reuse rates. Its core innovation is "online skill distillation" — after each successful task completion, the agent distills the execution trajectory into a reusable "skill" (a parameterized sub-policy) and directly invokes it in subsequent tasks instead of re-reasoning. On VisualWebArena, after the first task, PANDO reduces inference time for subsequent tasks by an average of 55%, with only a 2% drop in success rate.  
⚙️ Engineering Impact: This means building "self-improving with use" agent systems becomes feasible. When designing agent frameworks, teams should incorporate a "skill library" component and design trajectory-to-skill distillation strategies, rather than focusing solely on single-task inference optimization.

**[DynaFLIP: Rethinking Robotics Perception via Tri-Modal-Dynamics Guided Representation](https://huggingface.co/papers/2605.30350)**  
🔬 Breakthrough: Challenges the assumption that "robot vision encoders should be pre-trained on static image recognition." DynaFLIP proves that **prioritizing "motion understanding" in the pre-training phase of vision encoders is more effective than learning motion features from scratch in the policy network**. Its method pre-trains on "image-language-3D optical flow" triplets, enabling the encoder to infer object motion properties from a single image frame. On MetaWorld and Franka Kitchen benchmarks, policies using the DynaFLIP encoder achieve an average 18% higher task success rate and 25% better robustness to visual distractions compared to those using the CLIP encoder.  
⚙️ Engineering Impact: This means the robotics learning pipeline needs restructuring: the pre-training objective for vision encoders should shift from "recognizing objects" to "understanding how objects move." When building robot foundation models, teams should prioritize datasets containing motion annotations (e.g., Bridge Data v2) over static image datasets alone.

**[CONF-KV: Confidence-Aware KV Cache Eviction with Mixed-Precision Storage for Long-Horizon LLM](https://huggingface.co/papers/2605.24786)**
🔬 突破：推翻了一個常見假設——「KV cache 驅逐策略應基於歷史注意力或時間窗口」。CONF-KV 證明，**模型在每一步解碼時的「不確定性」（即 next-token 分佈的熵）是比歷史注意力更有效的驅逐訊號**。在 128K token 的長上下文推理中，相比 H2O（基於歷史注意力）和 StreamingLLM（基於時間窗口），CONF-KV 在保持相同困惑度的前提下，將 KV cache 記憶體佔用降低 40%（從 32GB 降至 19GB）。
⚙️ 工程影響：這意味著長上下文 LLM 推理的顯存瓶頸可以被顯著緩解，且無需修改模型架構。團隊在部署長文檔問答或程式碼庫分析服務時，可直接將 CONF-KV 整合到 vLLM 或 TensorRT-LLM 的排程器中，以降低單次推理的 GPU 成本。

**[PANDO: Efficient Multimodal AI Agents via Online Skill Distillation](https://huggingface.co/papers/2605.24785)**
🔬 突破：挑戰了「AI Agent 效率提升依賴更多推理時計算」的假設。PANDO 發現，**多模態 Web Agent 的 70% 以上推理時間浪費在三種可預測的低效模式上**：重複動作循環、隱藏的探索成本、低 prompt cache 複用率。其核心創新是「線上技能蒸餾」——Agent 在每次成功完成任務後，將執行軌跡提煉為一個可複用的「技能」（一個參數化的子策略），並在後續任務中直接呼叫而非重新推理。在 VisualWebArena 上，PANDO 在首次任務後，後續任務的推理時間平均降低 55%，且成功率僅下降 2%。

**[DynaFLIP: Rethinking Robotics Perception via Tri-Modal-Dynamics Guided Representation](https://huggingface.co/papers/2605.30350)**
